# BANK API

Simple Bank REST API which enables CRUD and making transfers features

### Tech Stack And Dependencies
- Java 11
- Spring Boot 2.7.1
- Spring Data JPA 2.7.1
- Spring Web 2.7.1
- Apache Maven 4.0.0
- Hibernate Validator
- Lombok 1.18.22
- MapStruct 1.5.2.Final
- Springfox Swagger UI 3.0.0
- MySQL5 Database
